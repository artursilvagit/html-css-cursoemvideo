Color, Range e File em Formulários

Color, Range e File

Notas Importantes **

Usando o input do tipo RANGE nós devemos atribuir mais 2 parâmetros a ele, o min="" e também o max="", são básicamente o valor mínimo e máximo que o range vai ter.

Também é importante lembras que ao usar o input do tipo "file", o formulário deve ser do método POST pois o método GET possui um limite de envio de dados (3 mil bytes, que equivale a 3 kilobytes).
