Tipos de inputs em Formulários

input para mêses do ano (type="month")

input para dia, mês e ano (type="date")

input para horário (type="time")

input para números (type="number")

input para texto (type="text")

Para colocarmos um valor predefinido dentro das caixas de input nós podemos usar o parâmetro "value" e adicionar o valor que queremos.
